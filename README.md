# Semester CGPA Calculator

A collection of C++ programs designed to help students calculate their semester-wise Grade Point Average (GPA). Each program is tailored for a specific semester, taking into account the subjects and their corresponding credit hours.

## üìú Description

This repository contains individual C++ files, each representing a semester's CGPA calculator. These command-line applications prompt the user to input their marks for various assessments like assignments, quizzes, mid-term exams, and final exams for each subject. Based on the provided marks, the program calculates the total percentage, grade points for each subject, and finally, the overall GPA for that semester.

## ‚ú® Features

* **Semester-wise Calculation:** Separate calculators for the 1st, 2nd, and 3rd semesters.
* **Detailed Mark Entry:** Input marks for assignments, quizzes, mids, and finals.
* **Automatic GPA Calculation:** Automatically calculates the grade points and the final GPA.
* **User-Friendly Interface:** Simple and interactive command-line interface.
* **Clear Result Display:** Shows a detailed table with subjects, percentages, credit hours, and grades.

## üöÄ How to Use


2.  **Navigate to the directory:**
    ```bash
    cd your-repository-name
    ```

3.  **Compile the desired semester's file:**

